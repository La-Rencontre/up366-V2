# up366-V2
基于up366项目改进的天学网答案提取，使用Gemini 3 Pro改进代码
原作者Github@EverNightCN B站@茯芺苒
[up366](https://github.com/EverNightCN/up366)
[茯芺苒的个人空间](https://space.bilibili.com/673274446)

主要改进：Pc.zip里包含的各小题js文件里有"T1-ZC.mp3"这类的信息，意思就是text1，总共有十段材料，但原版没有对这些材料本身进行排序，所以就会出现错序的问题，现已借助Gemini解决

---
## 使用说明
- 1.参考教程抓包到Pc.zip后解压
- 2.打开程序
- 3.选中questions文件夹的上一级文件夹（一般叫1或2）
- 4.提取

注：如果显示“未提取到答案”就换一个文件夹提取
